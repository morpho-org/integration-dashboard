# Morpho Integration Dashboard

A Next.js dashboard for Morpho integration and manual reallocation operations.

## Getting Started

### Prerequisites

- Node.js 18+
- Yarn

### Installation

```bash
yarn install
```

### Development

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Production Build

```bash
yarn build
yarn start
```

### Linting

```bash
yarn lint
```

## Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Wagmi / Viem
- RainbowKit
- Morpho SDK
